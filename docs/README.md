# bot-telegram — docs

Telegram driver for the togo bot subsystem (long-polling via go-telegram-bot-api).

- **Install:** `togo install togo-framework/bot-telegram`
- **Marketplace:** https://to-go.dev/marketplace
- **Source:** https://github.com/togo-framework/bot-telegram

See the repository [README](../README.md) for configuration and usage.
